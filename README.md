
# Snake Game

Snake Game made using Vanilla Js and deployed using Turbo 360 environment .
## Demo

https://snake-game-zkqph3.turbo360-vertex.com

  
## Deployment

To deploy this project run

```bash
  npm install
```
```bash
  turbo devserver
```

## Features

- Keyboard Controls
- Audio engagement
- Simple UI
